Thread It- Solo Project

Thread It is a project that was designed to mimic Reddit in style and functionality.

It was the first full stack project that I had created. It was built using a mobile first approach where the user could intuitively grasp the UI.

The project was written with a variety of tech, most notably Node.js, React.js, Express.js, PostgreSQL, HTML, & CSS.

Here is a link to the current hosted version. The current version only has suitable UI for mobile, so please use the dev tools to view it in mobile format.

Coroamer - Group Project

Coroamer is a mobile app that is designed to tackle the current travel restriction problem for travelers.

The pandemic has left a lasting impact on many lives and many people wish they could have a small break from the doom and gloom.

Coroamer's intention is to ease the finding of travel advice from governments by having it all in one location. Simply scroll to your destination and look up the restrictions.

The app is not yet hosted, but it can be viewed on GitHub by following this link. Here you can clone and pull the repo down.

To view the app, follow the instructions in the README.md and ensure you have expo or some emulator installed.
